# Индивидуальное Домашнее Задание: Анализатор Syslog
Добро пожаловать в документацию проекта! Данный инструмент предназначен для генерации, построчной обработки и категоризации логов системных событий

## Основные возможности:
* **Генерация тестовых данных:** Использование библиотеки `Faker`
* **Построчный анализ (Парсинг):** Эффективное чтение больших файлов через генераторы `yield`
* **Автоматический отчет:** Сохранение результатов в формате `JSON`

## Как запустить тесты:
python -m pytest
